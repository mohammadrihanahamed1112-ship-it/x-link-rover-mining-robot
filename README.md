# X-Link Rover: Adaptive Mining Tunnel Inspection & Gas Detection Robot

##  Description
The X-Link Rover is a robotic system designed for mining tunnel inspection. It can navigate narrow tunnels, detect harmful gases, and adjust its width using an X-link mechanism.

##  Objective
To improve miner safety by detecting hazardous gases and reducing human exposure in dangerous environments.

##  Components Used
- DC Motors
- Wheels and chassis
- Battery pack
- Motor driver (L298N)
- Gas sensors (MQ series)
- X-Link expansion mechanism
- ESP32

##  Working
The rover moves using motor-driven wheels. Gas sensors detect harmful gases like methane or carbon monoxide.  
The X-link mechanism expands or contracts to match tunnel width.

##  Features
- Gas detection system
- Adjustable width design
- Suitable for narrow tunnels
